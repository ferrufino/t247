Perfect Square


Description

  • Problem Description

    Is said that given a value N, the operation:

         N * [N +1] * [N +2] * [N +3] + 1

    generates a number that is a perfect square.
    Write a program that is useful to find the perfect square that can be obtained from the value provided by the user.

    Input

    An integer number.

    Output

    Perfect square is obtained from the number given as input. After displaying the number the cursor should be on the next line.

    Sample Input

    4

    Sample Output

    841

    Execution Example

    4
    841
